# Code of Conduct

This project adopts the [Contributor Covenant](https://www.contributor-covenant.org/), v2.1.

In short: be respectful, assume good faith, and keep discussion focused on the work.
Harassment, discrimination, and personal attacks are not tolerated. Maintainers may remove,
edit, or reject contributions and comments that violate these principles, and may ban
contributors for behavior they deem inappropriate.

Report concerns to the maintainers via a private channel listed in the repository profile.
The full Contributor Covenant text governs; see https://www.contributor-covenant.org/version/2/1/code_of_conduct/.
